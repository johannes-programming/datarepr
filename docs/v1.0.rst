Introduction
------------

This project allows to store the arguments for callables.

Installation
------------

To install datarepr, you can use pip. Open your terminal and run:

.. code-block:: shell

    pip install datarepr

Features
--------

``datarepr.datarepr(name: Any, /, *args: Any, **kwargs: Any) -> str``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function allows for common sense representation.

It is implemented as follows:

.. code-block:: python

    def datarepr(name: Any, /, *args: Any, **kwargs: Any) -> str:
        "This function allows for common sense representation."
        parts = list()
        for a in args:
            parts.append(repr(a))
        for i in kwargs.items():
            parts.append("%s=%r" % i)
        content = ", ".join(parts)
        ans = "%s(%s)" % (name, content)
        return ans
